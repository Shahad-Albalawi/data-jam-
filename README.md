# Real Estate Deals – Data Jam Challenge 2023

## Overview
This project analyzes real estate deals using Power BI. It calculates average prices per property type, counts properties by classification, analyzes areas per region, and summarizes total reference numbers.

## Features
- Clear overview of property types and average prices
- Interactive charts for property counts and area analysis
- Insightful metrics on total reference numbers

## Technologies
Power BI Desktop, Excel/CSV, Data Modeling

## How to Run
1. Make sure Power BI Desktop is installed.
2. Download the `RealEstateDeals.pbix` file.
3. Open it using Power BI Desktop to explore the dashboard.
